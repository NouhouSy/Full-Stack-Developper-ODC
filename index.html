Ma Galerie Responsive

Collection d'Images avec CSS Grid

Cette galerie utilise CSS Grid pour s'adapter à différentes tailles d'écran. Sur mobile, les images s'affichent en une seule colonne. Sur tablette (à partir de 772px), elles s'organisent en deux colonnes avec des bordures circulaires. Sur grand écran (à partir de 992px), la mise en page passe à trois colonnes avec des variations de taille.
